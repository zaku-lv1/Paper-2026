# 銀鏡反応シミュレーション

銀鏡反応における還元速度飽和現象を数理的に再現するために作成した、JavaScript製の数理シミュレーションである。

# 研究概要

本シミュレーションでは、銀イオンおよび還元剤を粒子として扱い、ブラウン運動・拡散律速・Langmuir型飽和・透過率変化を組み合わせることで、銀膜形成過程を可視化している。

銀鏡反応では、還元剤濃度を上げると析出速度は増加するが、一定以上では速度増加が頭打ちになる「飽和現象」が確認される。

本研究では、

- 粒子拡散
- 壁面近傍での反応
- Langmuir型速度飽和
- Michaelis-Menten型近似
- ランベルト・ベール則

を用いて、この現象を数理的に再現した。

# シミュレーションで再現している要素

## 1. ブラウン運動

粒子はランダムウォークを行う。

```js
this.vx += (Math.random()-0.5)*0.06;
this.vy += (Math.random()-0.5)*0.06;
```

これにより、溶液中の熱運動を簡略化して表現している。

---

## 2. 壁面近傍での銀析出

銀鏡反応はガラス壁面で起こるため、粒子が壁面付近に存在するときのみ反応判定を行っている。

```js
const nearWall =
    p.x < 10 + WALL_MARGIN ||
    p.x > 410 - WALL_MARGIN ||
    p.y < 10 + WALL_MARGIN ||
    p.y > 410 - WALL_MARGIN;
```

---

## 3. 反応速度飽和

高濃度条件で速度が無限に増加しないよう、Langmuir型飽和式を導入している。

反応速度係数：

```math
k_{eff}=k_{base}\times\frac{N_{red}}{K_M+N_{red}}
```

反応確率：

```math
P=1-e^{-k_{eff}\Delta t}
```

これにより、

- 低濃度では濃度依存的に高速化
- 高濃度では飽和

という挙動を再現している。

---

## 4. 透過率変化

銀析出量から透過率を指数関数的に計算している。

```math
T=100e^{-\gamma N_{silver}}
```

銀膜形成による光遮蔽を可視化している。

# 既知の問題

## 1. 高負荷時の処理落ち

粒子数が増えるとブラウザ描画負荷が上昇する。

特に：

- 高濃度
- 高フレームレート
- 古いPC

では処理速度が低下する場合がある。

---

## 2. 粒子探索計算量

現在は全粒子同士を探索している。

```js
for(let i=0;i<particles.length;i++){
    for(let j=0;j<particles.length;j++){
```

計算量は O(N²) であり、粒子数増加時に負荷が大きくなる。

# 今後の実装予定

## 速度定数のキャリブレーション

先行研究の実験結果とシミュレーション結果がより一致するよう、

- k_base
- K_M

などのパラメータを調整予定。

---

## 温度パラメータ

アレニウス式：

```math
k(T)=Ae^{-\frac{E_a}{R(T+273.15)}}
```

を導入し、温度による反応速度変化を再現予定。

また、粒子速度 `vmax` を温度依存にすることで、加熱条件下での拡散速度変化も再現予定。
